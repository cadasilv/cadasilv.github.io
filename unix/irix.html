Sistema Unix SGI IRIX

HP-UX é uma pronúncia abreviada de Hewlett Packard Unix. Este sistema operacional é definido pelo UNIX System V versão 4, que se tornou o centro das atenções em 1984. Seu objetivo de desenvolvimento original visava o PC Proprietary Integral da HP. Posteriormente, foi introduzido nas plataformas de hardware dos servidores de negócios da série 9000.

Sob uma permissão de acesso a arquivos do sistema operacional, o HP-UX introduziu a funcionalidade de lista de controle de acesso para todos os usuários. Este sistema operacional possui algumas características corporativas representadas por uma variedade de ferramentas e funcionalidades integradas em seu núcleo. É famoso por seus impressionantes recursos de gerenciamento de segurança e pela flexibilidade e disponibilidade de seu gerenciamento de memória

O HP-UX está equipado com uma variedade de ferramentas para uso em empresas para monitoramento (custos, carga de trabalho), bem como para registro e visualização. Ao lado da versão base, há uma Enterprise Edition com gerenciamento de recursos e a Mission Critical Edition para empresas com aplicativos críticos para maior disponibilidade e segurança. Este sistema operacional possui uma detecção de invasão de host (H-IDS) integrada. Este sistema operacional é usado para servidores de conteúdo, servidor web, servidor dedicado, bancos de dados ou também para sistemas de cluster.

O VUE (Visual User Environment) e o CDE estão disponíveis como uma GUI, a última nomeada é a GUI padrão desde o HP-UX versão 10.20. A HP desenvolve o HP-UX 11i versão 4 e planeja o HP-UX 11i versão 5 como uma meta de longo prazo. HP-UX 11i versão 3 ainda estará pronto no outono de 2006. A plataforma PA-RISC será cancelada para 2012, a migração para sistemas Itanium HP será realizada.

Download e Instalação

HP-UX 10.20 , 11.00 , 11.11 podem ser instalados no hardware HP9000 e Itanium versão 11i ver2 pode ser instalado em sistemas baseados em Itanium. Não há downloads nem instalação em regular Pcs.
