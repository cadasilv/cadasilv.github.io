Sistema Unix Microsoft/SCO Xenix

O sistema operacional baseado no Xenix Unix estava sob uma licença da Microsoft quando foi descontinuado durante a compensação do final dos anos 1970. A popularidade do UNIX mais tarde impressionou a Microsoft, atraindo-a para comprar os direitos de licença da AT&T. Ainda não é de conhecimento comum que os direitos do UNIX já estiveram sob as asas da Microsoft. No entanto, a Microsoft não estava diretamente envolvida na comercialização do Xenix para clientes específicos.

A licença comercial foi concedida a empresas como Intel, IBM e SCO, cujos processadores proprietários foram usados ​​para portar o sistema operacional Unix. Seu rompimento com a AT&T removeu a vantagem competitiva da Microsoft. Posteriormente, a Microsoft transferiu seus direitos UNIX para a SCO, que era uma parte interessada. A SCO então trabalhou em um lançamento agora do Xenix e o renomeou para SCO UNIX. No entanto, o SCO UNIX V.2.3.4 de 1991 foi a última versão lançada, pois não conseguiu avançar no desenvolvimento.

Xenix/SCO UNIX System V – um sistema operacional da família UNIX de origem fechada publicado em 1980 pela Microsoft e depois vendido para a Santa Cruz Operation (SCO). Microsoft(R) XENIX(R) System V/386 foi o primeiro lançamento da implementação da Microsoft do sistema operacional UNIX(R) System V para o microprocessador Intel(R) 80386.

O XENIX é uma solução multiusuário que permite que vários usuários sejam conectados por meio de terminais baratos a uma única máquina, permitindo assim que os usuários compartilhem os recursos da máquina. Também é compatível com o Sistema Operacional UNIX System V – um sistema operacional altamente portátil, encontrado em uma ampla variedade de arquiteturas: grandes mainframes, minicomputadores, estações de trabalho técnico-científicas e computadores pessoais.

A partir de 1981, a Microsoft comercializou sua adaptação do sistema operacional UNIX sob o nome de XENIX. O objetivo da Microsoft era fornecer implementações de alta qualidade do UNIX para sistemas de computador baseados em microprocessadores. Ao fazer isso, a Microsoft tornou-se um dos principais licenciados do sistema operacional UNIX, respondendo pela maioria das implementações vendidas em sistemas baseados em microprocessador. Em particular, uma grande base de aplicativos foi criada em torno do Microsoft XENIX System V/286 para sistemas baseados no microprocessador Intel 80286. O Microsoft XENIX System V/386 para o Intel 80386 preservará essa base de aplicativos, abrindo todo o potencial do 80386 para desenvolvedores e usuários.
