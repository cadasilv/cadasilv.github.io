Sistema Unix TRU64

É o Sistema Operacional derivado do UNIX da Digital Equipment Corporation. Foi descontinuado enquanto ainda tinha raízes de desenvolvimento para Alpha ISA (Instruction Set Architecture). O sistema operacional OSF/1 inspirou a criação do Tru76 sob a Open Software Foundation (OSF). OSF tinha o objetivo de desenvolvimento de competir com o lançamento do UNIX System V. Atualmente, a Hewlett Packard adquiriu o Tru64 como produto e marca registrada. O Tru64 UNIX 5.0 facilitou o recurso de agrupamento por meio de um TruCluster Server.

A maioria de seus recursos, como TruCluster, AdvFS e LSM, migrou para HP-UX, um produto UNZIX carro-chefe da HP. Dezembro de 2012 foi o suporte de fim de vida para o Tru64 da HP.

Histórico do produto

Tru64 UNIX é a implementação Compaq da tecnologia Open Software Foundation Versão 1.0 e Versão 1.2 e da interface gráfica do usuário e ambiente de programação Motif Versão 1.2.5. Além disso, o Tru64 UNIX suporta todos os recursos do sistema X Window, versão 11, versão 6.3 (X11R6.3). O sistema operacional Tru64 UNIX é uma arquitetura de kernel avançada multiusuário/multitarefa de 64 bits baseada no design de kernel Mach versão 2.5 da Carnegie Mellon University com componentes da Berkeley Software Distribution (BSD) versões 4.3 e 4.4, UNIX System Laboratories System V Release 4.0, outras fontes de software, de domínio público e da Compaq Computer Corporation. As versões anteriores do Tru64 UNIX eram conhecidas como DIGITAL UNIX e, em sua oferta inicial, DEC OSF/1. 1.1.2 Descrição do Produto Tru64 UNIX é suportado em todos os servidores Alpha e estações de trabalho com 64 MB de memória. O sistema operacional incorpora vários aprimoramentos de desempenho desenvolvidos ou estendidos pela Compaq, incluindo memória com fio, memória virtual e Unified Buffer Cache; agrupamento de blocos de arquivos do sistema de arquivos UNIX e gravações em cache sobre NFS; IP Multicasting, descoberta de MTU de caminho e TCP/IP otimizado; e bibliotecas compartilhadas de início rápido.

Com o Tru64 UNIX, você tem um ambiente de administração de sistema claro e conciso (incluindo gráficos, baseados na web e células de caracteres) que: Simplifica as tarefas de administração do sistema Permite uma instalação de atualização que não sobrescreve os arquivos do sistema ou executa uma instalação completa que coloca seu sistema em funcionamento rapidamente Oferece suporte a drivers carregáveis ​​e outros subsistemas do kernel, incluindo suporte de caminho de inicialização carregável para discos de terceiros e placas gráficas Fornece gerenciamento virtualmente automático de dispositivos periféricos de armazenamento SCSI, como discos, fitas e CD-ROMs Tru64 UNIX suporta o Common Desktop Environment (CDE) como a interface de usuário padrão. O CDE fornece uma interface gráfica de usuário uniforme, portátil em várias plataformas, para facilitar as tarefas comuns de administração do sistema e do usuário final.

O Tru64 UNIX fornece suporte em tempo real e multiprocessamento simétrico (SMP), servidores e clientes sem dados e vários recursos para auxiliar os programadores de aplicativos no desenvolvimento de aplicativos que usam bibliotecas compartilhadas, threads e arquivos mapeados na memória. É totalmente compatível com a especificação Single UNIX, com a marca X/Open UNIX, com POSIX 1003.1b (tempo real) e com POSIX 1003.1c (com POSIX Threads -- anteriormente chamados de DECthreads).
